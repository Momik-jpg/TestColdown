# Prüfungs-Countdown App

## App-Anleitung (Handy)
1. Öffne das Repository: `https://github.com/Momik-jpg/TestColdown`
2. Klicke oben auf `Actions`.
3. Wähle den Workflow `Build Android APK`.
4. Klicke rechts auf `Run workflow`.
5. Wähle Branch `main` und klicke nochmal `Run workflow`.
6. Warte bis der Lauf `Success` ist.
7. Scrolle im Workflow-Lauf nach unten zu `Artifacts`.
8. Lade `ExamCountdown-debug-apk` herunter.
9. Entpacke die ZIP-Datei.
10. Installiere `app-debug.apk` auf deinem Handy.

## Hinweis
- Auf Android musst du eventuell `Unbekannte Apps installieren` erlauben.
